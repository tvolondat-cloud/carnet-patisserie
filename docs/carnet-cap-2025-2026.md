## Section 3 — Biscuits et génoises

### Génoise (EP2)
| Ingrédient | Quantité |
|---|---|
| Œufs entiers | 6 u (300 g) |
| Sucre | 180 g |
| Farine T55 | 180 g |
| Beurre fondu | 50 g |

Monter les œufs entiers avec le sucre au bain-marie jusqu'à 50°C, puis foisonner hors du feu jusqu'au ruban. Incorporer farine tamisée par mouvements enveloppants, puis beurre fondu tiède. Cuire 20–25 min à 175°C.

## Techniques clés CAP

| Technique | Description |
|---|---|
| Sablage | Enrobage de la farine par la matière grasse avant ajout du liquide |
| Fraser | Pousser la pâte avec la paume pour homogénéiser sans développer le gluten |
| Dessécher | Cuisson de la panade à choux pour éliminer l'humidité |
| Tourage | Stratification beurre/détrempe par pliages successifs |
| Foisonner | Incorporer de l'air en fouettant (crème, beurre) |
| Foncer | Chemiser un cercle ou moule avec la pâte |
| Chiqueter | Marquer le bord d'une abaisse avec la pince ou les doigts |
| Blanchir | Fouetter jaunes + sucre jusqu'à ruban clair |
| Nappe | Test de cuisson de la crème anglaise (trait net sur spatule) |
| Ruban | Appareil qui retombe de façon lisse et continu du fouet |
| Filmage au contact | Filmer directement sur la surface pour éviter la peau |
| Tempérer | Mettre le chocolat aux bonnes températures de travail |
| Chablonner | Enduire un fond de tarte de chocolat fondu pour l'imperméabiliser |
| Chemiser | Tapisser un moule ou cercle de biscuits ou de pâte |
| Imbiber | Humidifier un biscuit de sirop aromatisé |
| Apprêt | Deuxième pousse des viennoiseries avant cuisson |
| Pointage | Première pousse d'une pâte levée |
